Feature request as install won't work without --cert-path, --fullchain-path or --key-path parameters:
./certbot-auto install --noninteractive --apache --domains pcm.4delphi.com
But run (which is the default command) will request and install without any extra parameters:
./certbot-auto --noninteractive --apache --domains pcm.4delphi.com
And certonly will request without any extra parameters:
./certbot-auto certonly --noninteractive --apache --domains pcm.4delphi.com
So please make install as smart as certonly or (default) run that somehow figure out the correct locations of the certificates.
