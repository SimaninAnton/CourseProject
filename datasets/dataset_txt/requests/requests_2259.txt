The HTTP proxy support in the current release (1.0.3) is broken.
A valid request sent to the proxy would be:
GET http://google.de/ HTTP/1.1\r\n-
Requests, however, does this:
GET / HTTP/1.1\r\n-
Obviously, this won't work. Previous versions (0.14, for example) are not affected.
