In section Binary Response Content, r.content returns an instance of bytes while StringIO is an in-memory stream for text only. It should use BytesIO instead.
from PIL import Image
from io import BytesIO

i = Image.open(BytesIO(r.content))
Use original code will raise TypeError: initial_value must be str or None, not bytes.
