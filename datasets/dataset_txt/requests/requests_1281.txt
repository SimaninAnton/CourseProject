It would be good if were able to pass the 'no_proxy' value in the proxies dictionary argument. Currently it appears have to set the environment variable 'no_proxy'.
For example:
proxies = {'https_proxy': 'http://proxy.example.com:3128',
           'no_proxy': '10.0.0.0/16,example.com,.example.com'}
requests.get('http://www.example.com', proxies=proxies)
