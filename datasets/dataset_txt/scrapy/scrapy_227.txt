Please take a look at this code:
import scrapy
from scrapy.loader import ItemLoader


class MyItem(scrapy.Item):
    a = scrapy.Field()


class MyLoader(ItemLoader):
    item = MyItem()


loader = MyLoader()
loader.add_value("a", "b")
item = loader.load_item()
print(item.get("a"))

loader = MyLoader()
loader.add_value("a", [])
item = loader.load_item()
print(item.get("a"))
The output should be something like
['b']
None
But it actually is
['b']
['b']
