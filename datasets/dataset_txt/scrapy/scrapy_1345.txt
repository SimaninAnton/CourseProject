In the file shell.py I propose a change in this function "inspect_response":
def inspect_response(response, spider):
    """Open a shell to inspect the given response"""
    Shell(spider.crawler).start(response=response)
This is my proposal:
def inspect_response(response, spider):
    """Open a shell to inspect the given response"""
    Shell(spider.crawler).start(response=response, spider=spider)
Rationale: Without this change, when you call to the function inspect_response(), you obtain a None object inside the "spider" variable. None is the default value for the argument "spider" of the "start()" method of the Shell class.
