"connect_timeout" and "request_timeout" values are required in request for WebSocketClientConnection connection, but not documented in http://www.tornadoweb.org/en/stable/websocket.html#client-side-support
Without these timeout values getting 'unorderable types: NoneType() < NoneType()' error with this code below;
    request = httpclient.HTTPRequest(
        url = "wss://...",
        method = "GET",
        validate_cert = False,
        client_key = "/ca/client-key.pem",
        client_cert = "/ca/client-cert.pem")
        #connect_timeout = 10,
        #request_timeout = 10)
    conn = websocket.WebSocketClientConnection(ioloop.IOLoop.current(), request)
    conn.connect_future.add_done_callback(conncallback)
