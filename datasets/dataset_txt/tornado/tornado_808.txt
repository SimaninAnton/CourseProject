It is common that template file consist of empty lines, for example the websocket demo in Tornado package:
<div id="inbox">
    {% for message in messages %}
        {% include "message.html" %}
    {% end %}
</div>
Tornado currently produces this:
<div id="inbox">

<div class="message" id="m8769fcc8-57ee-4f4e-814d-bcfbae0c8598">message1</div>


<div class="message" id="me329adf7-9811-46b4-a332-d4a3e4261b4a">message2</div>


</div>
As you see, there are 5 empty lines.
I suggest that empty lines should be compressed along with spaces when compress_whitespace is set.
