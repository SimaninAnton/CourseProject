Was wondering what to do, to structure a Tornado app when running it with Jupyter. The test app below works fine from the Python command line and code changes using vim will register. But if I load this into notebook cell although it runs, changes do not register unless I restart the Kernel.
import tornado.ioloop
import tornado.web

class MainHandler(tornado.web.RequestHandler):
    def get(self):
        self.write("<b>Hello Whats Up, world!!!<br>Test this</b>")

def make_app():
    return tornado.web.Application([
        (r"/", MainHandler),
    ],debug=True)

if __name__ == "__main__":
    port=8889
    app = make_app()
    app.listen(port)
    print(f'listening on port {port}')
    tornado.ioloop.IOLoop.current().start
