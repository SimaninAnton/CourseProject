When trying to setup a tornado ioloop inside an asyncio eventloop, when the eventloop is a ProactorEventLoop you get a NotImplementedException when you try to hook up some socket-related component since ProactorEventLoop does not implement add_reader.
The following code raises that exception (on Windows):
import asyncio
import socket, os
from tornado.platform.asyncio import AsyncIOMainLoop
from tornado.iostream import IOStream

if os.name == 'nt':
    loop = asyncio.ProactorEventLoop() # for subprocess' pipes on Windows
    asyncio.set_event_loop(loop)
else:
    loop = asyncio.get_event_loop()
AsyncIOMainLoop().install()

i = IOStream(socket.socket())
i.connect(('www.google.com', 80), lambda: None)
