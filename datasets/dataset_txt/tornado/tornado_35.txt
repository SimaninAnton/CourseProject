Consider the Hello World example:
import tornado.ioloop
import tornado.web

class MainHandler(tornado.web.RequestHandler):
    def get(self):
        self.write("Hello, world")

def make_app():
    return tornado.web.Application([
        (r"/", MainHandler),
    ])

if __name__ == "__main__":
    PORT = 8080
    ADDR = '192.168.100.0'
    app = make_app()
    app.listen(PORT, ADDR)
    tornado.ioloop.IOLoop.current().start()
For ADDR set to '0.0.0.0' or '127.0.0.1', I am able to access the webpage from localhost:8080. However, if I try to listen only on my LAN, '192.168.100.0' doesn't do anything and '192.168.100.0/24' is just a parse error.
Is there any reasonable way to listen to an IP address range? (Particularly for LAN.)
