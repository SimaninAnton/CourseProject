In version 4.1, when a new IOLoop is created, it automatically becomes “current” for the thread if there is not already a current instance.
so, if I instance an IOLoop before application.listen(8888), HttpServer will add this socket into the IOLoop I instance before. finally I run "tornado.ioloop.IOLoop.instance().start()", socket is NOT added into this "main loop".
import tornado.ioloop
import tornado.web

_ioloop = tornado.ioloop.IOLoop()

class MainHandler(tornado.web.RequestHandler):
    def get(self):
        self.write("Hello, world")

application = tornado.web.Application([
    (r"/", MainHandler),
])

if __name__ == "__main__":
    application.listen(8888)
    tornado.ioloop.IOLoop.instance().start()
