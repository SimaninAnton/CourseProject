Every time server receives message from browser using WebSockets, I have to rebind on_message method again to receive future messages, like this:
class MessageHandler(websocket.WebSocketHandler):
def open(self):
self.receive_message(self.on_message)
def on_message(self, message):
    # process message 
    self.receive_message(self.on_message)
