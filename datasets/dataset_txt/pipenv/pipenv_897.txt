If resolver.py is called in verbose and/or debug mode, it won't log certain messages, such as the warning when the PKG-INFO file is not found. This is because the logger set to INFO or DEBUG is nonpip, while that message is logged to pipenv.patched.notpip._internal.req.req_install, whose level is ERROR anyway.
