Issue description
When using run to run a command, pipenv is unable to run commands that includes flags when they are in quotes, rather than using --
pipenv run "python -m pytest"
Expected result
Python to run pytest.
Actual result
Error: the command python -m pytest could not be found within PATH or Pipfile's [scripts].
Steps to replicate
pipenv run "python" - runs fine
pipenv run "python -m" - error
pipenv run -- python - runs fine
pipenv run -- python -m -runs fine
$ pipenv --support
