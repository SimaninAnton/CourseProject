Problem Description
When running 'brew install mitmproxy' it have dependency to just python without specifying Python2 or Python3, resulting that Python2 is removed on Mac OS Catalina.
Steps to reproduce the behavior:
brew install mitmproxy
Try to run 'python2' afterward
Solution
Change dependency from 'python' to 'python3' as described where:
https://docs.brew.sh/Homebrew-and-Python
Where to change:
https://github.com/Homebrew/homebrew-core/blob/master/Formula/mitmproxy.rb
https://formulae.brew.sh/api/formula/mitmproxy.json
