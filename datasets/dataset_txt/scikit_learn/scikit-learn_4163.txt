https://github.com/scikit-learn/scikit-learn/blob/master/sklearn/feature_selection/rfe.py#L129
Why in RFE class the step value is rounded down and can be 0?
step = int(self.step * n_features)
I think it should be rounded down but with a minimum value of 1.
step = [1, n_features-1]
Example:
from sklearn.svm import SVR
from sklearn.feature_selection import RFE
from sklearn.datasets import make_friedman1
X, y = make_friedman1(n_samples=50, n_features=10, random_state=0)
estimator = SVR(kernel="linear")
selector = RFE(estimator, step=0.01)
selector = selector.fit(X, y)
Raising this error:
ValueError: Step must be >0
