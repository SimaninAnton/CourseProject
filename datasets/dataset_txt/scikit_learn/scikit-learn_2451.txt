Description
Local Outlier Factor (LOF) raises a segmentation fault when applied to large datasets (e.g. 100 000 samples).
Running the following code requires at least 10GB RAM and will run for at least 30 minutes.
Steps/Code to Reproduce
import numpy as np
from sklearn.neighbors import LocalOutlierFactor

np.random.seed(0)
X = np.random.uniform(size=(100000, 2))
clf = LocalOutlierFactor(len(X) * 0.1)
Y = clf.fit_predict(X)
Expected Results
No segfault.
Actual Results
7904 segmentation fault (core dumped)
