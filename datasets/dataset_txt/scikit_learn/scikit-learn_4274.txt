On the documentation page: http://scikit-learn.org/stable/modules/generated/sklearn.manifold.MDS.html
When describing the fit() and fit_transform() function, currently it is:
X : array, shape=[n_samples, n_features]
    Input data.
init : {None or ndarray, shape (n_samples,)}, optional
    If None, randomly chooses the initial configuration if ndarray, initialize the SMACOF algorithm with this array.
It'd be helpful to note that if dissimilarity='euclidean', one would make X as [n_samples, n_features]. However, if dissimilarity='precomputed', then a dissimilarity matrix of [n_samples, n_samples] should be fed into the estimator. Suggested wording:
X : array, shape=[n_samples, n_features] dissimilarity is "euclidean", or an [n_samples, n_samples] dissimilarity matrix if dissimilarity is "precomputed".
    Input data.
init : {None or ndarray, shape (n_samples,)}, optional
    If None, randomly chooses the initial configuration if ndarray, initialize the SMACOF algorithm with this array.
