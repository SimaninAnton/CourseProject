Description
On 32 bit versions of Windows, when min_max_axis is called on a csc matrix where indptr.dtype is int64, an error is produced. This prevents this pull request passing tests (see here).
Steps/Code to Reproduce
import scipy.sparse as sp
from sklearn.utils.sparsefuncs import min_max_axis

X = sp.csc_matrix([[1,2],[3,4]])
X.indptr = X.indptr.astype('int64')

Y = sp.csr_matrix([[1,2],[3,4]])
Y.indptr = Y.indptr.astype('int64')

print(min_max_axis(Y, 0))
print(min_max_axis(X, 0))
