Description
When n_components is set to be a value greater than n_features, then n_components will default to the same value as n_features (i.e. n_components = n_features).
Additionally, not all of the n_components = n_features components will be learned.
Steps/Code to Reproduce
from sklearn.decomposition import FactorAnalysis
import numpy

X = numpy.random.randn(1000,9)

fa = FactorAnalysis(n_components=12)
fa.fit(X)
print('FactorAnalysis components_ shape: {}'.format(fa.components_.shape))
Expected Results
FactorAnalysis components_ shape: (9, 12)
Actual Results
FactorAnalysis components_ shape: (9, 9)
