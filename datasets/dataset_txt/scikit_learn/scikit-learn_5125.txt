Image denoising example on the web page:
http://scikit-learn.sourceforge.net/dev/auto_examples/decomposition/plot_image_denoising.html
Has two issues:
Wrong argument name:
dico = MiniBatchDictionaryLearning(n_components=100, alpha=1, n_iter=500)
should be:
dico = MiniBatchDictionaryLearning(n_atoms=100, alpha=1, n_iter=500)
(i.e. n_atoms instead of n_components.)
Logical bug:
print 'Extracting clean patches...'
...
data = extract_patches_2d(distorted[:, :height / 2], patch_size)
The patches should be clean, not distorted! (amazingly, this still works, due to the dictionary robustness). therefore:
data = extract_patches_2d(lena[:, :height / 2], patch_size)
is preferable.
