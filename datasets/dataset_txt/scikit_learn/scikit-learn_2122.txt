Running the example here:
https://github.com/scikit-learn/scikit-learn/blob/master/examples/cluster/plot_dbscan.py
throws an error with the message:
ValueError: The truth value of an array with more than one element is ambiguous. Use a.any() or a.all()
The cause is the color specification. The example uses a list, which is not one of the ways that colors can be specified in matplotlib. See https://matplotlib.org/users/colors.html.
My system uses Python 3.5, matplotlib 2.0.2, scipy 0.19.0, numpy 1.13.0, and scikit-learn 1.18.1
