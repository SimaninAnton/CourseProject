Description
sklearn.datasets.make_regression accepts the optional coef argument to return the coefficients of the underlying linear model. This is useful for testing models by comparing estimated coefficients to the ground truth. Analogously, sklearn.datasets.make_classification should optionally return a boolean array of length [n_features] indicated whether or not a feature is useful.
Steps/Code to Reproduce
from sklearn.datasets import make_classification

X, y, idx = make_classification(
    n_samples=50, n_features=20, n_informative=4,
    n_redundant=0, n_repeated=0, n_classes=2, useful_indices=True
)

print(idx)
Expected Results
A boolean array, where a True element indicates that the corresponding feature is either
informative, redundant, or repeated.
[False  True False False False False  True False False False False  True
 False False False False False  True False False]
