To use svmlight for multi-label classification, the following format will not work :
324, 674, 234, 54 234:1 765:3 543:10
It has to be in the following format :
324,674,234,54 234:1 765:3 543:10
If there are spaces between labels along with 'commas' , this command gives a ValueError.
