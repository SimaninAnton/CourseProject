PCA().fit_transform() apparently changes c-order arrays into f-order arrays:
import numpy as np
from sklearn.decomposition import PCA

x = np.random.uniform(-1, 1, size=(10, 10))
print(x.flags.c_contiguous)

y = PCA().fit_transform(x)
print(y.flags.c_contiguous)
prints True // False
