In the SVR class, passing a transposed matrix into the fit() function causes a 'ValueError: ndarray is not C-contiguous' message. The same error does not happen with other regression classes, e.g. LinearRegression.
This post at stackoverflow may be relevant? http://stackoverflow.com/questions/4420622/how-to-account-for-column-contiguous-array-when-extending-numpy-with-c
Some example code follows:
import numpy
import scikits.learn.linear_model
import scikits.learn.svm
X = numpy.random.random(15)
X = numpy.reshape(X, (3,5))
X = X.T
y = numpy.random.random(5)
lin = scikits.learn.linear_model.LinearRegression()
svr = scikits.learn.svm.SVR()
lin.fit(X, y)
lin.predict(X)
svr.fit(X, y)
svr.predict(X) #causes ValueError
