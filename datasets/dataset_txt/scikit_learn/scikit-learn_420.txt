What is the motivation of using @property for hiding unavailable function methods:
import numpy as np
from sklearn.linear_model import SGDClassifier

X = np.array([[-1, -1], [-2, -1], [1, 1], [2, 1]])
Y = np.array([1, 1, 2, 2])
clf = SGDClassifier(max_iter=1000, tol=1e-3)
clf.fit(X, Y)

# Case 1
# raises AttributeError
clf.predict_proba

# Case 2
# also raises AttributeError
clf.predict_proba(X)
Using @property allows for Case 1 to raise an error.
