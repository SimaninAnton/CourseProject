Description
LDA and QDA do not accept (scipy) sparse matrix.
Steps/Code to Reproduce
import numpy as np
from scipy.sparse import lil_matrix
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis, QuadraticDiscriminantAnalysis

x = lil_matrix(np.random.rand(5,3))
y = [1, 1, 1, 0, 0]

clf_lda = LinearDiscriminantAnalysis()
clf_qda = QuadraticDiscriminantAnalysis()

clf_lda.fit(x, y)  # doesn't work
clf_qda.fit(x, y)  # doesn't work

clf_lda.fit(x.todense(), y)  # works
clf_qda.fit(x.todense(), y)  # works

clf_lda.predict(x, y)  # (surprisingly) works
clf_qda.predict(x, y)  # doesn't work
Is this a bug or an unimplemented feature?
