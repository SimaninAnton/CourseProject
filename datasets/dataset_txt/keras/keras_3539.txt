in keras/objectives.py
I replaced
def get(identifier):
    return get_from_module(identifier, globals(), 'objective')
with:
def get(identifier):
    if callable(identifier):
        return identifier
    return get_from_module(identifier, globals(), 'objective')
now when I call model.compile() I can pass it my own loss function and use it while training instead of the predefined ones in objectives.py
thought I would share to help others
