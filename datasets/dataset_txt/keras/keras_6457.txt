Running example imdb_bidirectional_lstm.py returns the error - TypeError: fit() got an unexpected keyword argument 'show accuracy'. Looks like the current implementation of the graph.fit() function in in models.py doesn't support the show_accuracy parameter. Modifying line 60 in imdb_bidirectional_lstm.py from
model.fit({'input': X_train, 'output': y_train}, batch_size=batch_size, nb_epoch=4, show_accuracy=True)
to
model.fit({'input': X_train, 'output': y_train}, batch_size=batch_size, nb_epoch=4)
fixes the issue.
