From the documentation at https://keras.io/layers/convolutional/. They suggest that the input for the convolution should follow the order: channels, rows, cols. However, when I run the example codes:
model = Sequential()
model.add(Convolution2D(64, 3, 3, border_mode='valid', input_shape=(3, 256, 256)))
print model.output_shape # output (None, 1, 254, 64)
I get (None, 1, 254, 64) instead of (None, 64, 254, 254). When I switch input shape to: rows, cols, channels, I got the correct result.
model = Sequential()
model.add(Convolution2D(64, 3, 3, border_mode='valid', input_shape=(256, 256, 3)))
print model.output_shape # output (None, 254, 254, 64)
I use Keras version: 1.1.2 (updated from master branch).
