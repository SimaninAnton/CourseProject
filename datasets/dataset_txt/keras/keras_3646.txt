Using K.dot(a,b) in the tensorflow backend to compute a vector/matrix dot product crashes because of different dims. The same calculation in the theano backend is correctly computed.
Similarly, matrix/vector and vector/vector products fail
Gist
Check that you are up-to-date with the master branch of Keras. You can update with:
pip install git+git://github.com/fchollet/keras.git --upgrade --no-deps
If running on TensorFlow, check that you are up-to-date with the latest version. The installation instructions can be found here.
If running on Theano, check that you are up-to-date with the master branch of Theano. You can update with:
pip install git+git://github.com/Theano/Theano.git --upgrade --no-deps
