Steps to reproduce
Have a model, serializer, view, router as:
class MyModel(models.Model):
    name = models.CharField(max_length=100, primary_key=True)

class MyModelSerializer(serializers.ModelSerializer):
    class Meta:
        model = MyModel
        fields = '__all__'

class MyModelViewSet(viewsets.ModelViewSet):
    queryset = MyModel.objects.all().order_by('name')
    serializer_class = MyModelSerializer

router = DefaultRouter()
router.register(r'mymodel', MyModelViewSet)
urlpatterns = patterns('',
    url(r'^api/', include(router.urls)),
)
Now PUT {'name': 'new-name'} to /api/mymodel/some-existing-name
Expected behavior
Existing model instance is renamed.
Actual behavior
A new model named new-name is created and the existing some-existing-name is ignored.
