Add a subclass of DjangoObjectPermissions with view permission:
class DjangoReadObjectPermissions(permissions.DjangoObjectPermissions):
    """
    Add checks for view permission on data "viewing".
    """
    perms_map = dict(permissions.DjangoObjectPermissions.perms_map.items()
                     + {
        u'GET': ['%(app_label)s.view_%(model_name)s'],
        u'HEAD': ['%(app_label)s.view_%(model_name)s'],
        u'OPTIONS': ['%(app_label)s.view_%(model_name)s']
    }.items())
Currently DjangoObjectPermissionsFilter already filters based on view permission.
