Similar to #467.
default is a attribute on model classes, which can be used for generating nice forms with preset values.
rest_framework.fields.Field and ModelSerializer should support it.
An idea to implement it
Add the condition in get_field_kwargs like:
if model_field.default:
    kwargs['default'] = model_field.default
Then in the SimpleMetadata add the default attr into the list of attrs:
attrs = [
    'read_only', 'label', 'help_text',
    'min_length', 'max_length',
    'min_value', 'max_value', 'default'
]
