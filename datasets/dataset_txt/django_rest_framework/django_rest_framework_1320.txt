If I need extra context out of serializer for validators documentation suggest me to do:
class MyValidator(object):
    def set_context(self, serializer):
        self.some_state = True

    def __call__(self, data):
        return True
Looking at source code I see that validators have been made exempt from deepcopy, and comment misleadingly says that validators are immutable:
https://github.com/tomchristie/django-rest-framework/blob/master/rest_framework/fields.py#L582
I validated that state persist globally doing this:
>>> class MySerializer(serializers.Serializer):
    class Meta:
        validators = [MyValidator()]
>>>  MySerializer(data={}).is_valid()
True
>>> MySerializer.Meta.validators[0].some_state
True
