In a simple RetrieveUpdateDestroyAPIView like so:
class ItemView(generics.RetrieveUpdateDestroyAPIView):
    serializer_class = PlaceSerializer
    queryset = Item.objects.all()
    authentication_classes = (SessionAuthentication, BasicAuthentication)
    permission_classes = (
        permissions.IsAuthenticated,
        HasItemPermission,
    )
with a custom permission class like so:
class HasItemPermission(permissions.BasePermission):
    def has_object_permission(self, request, view, obj):
        print(True)
        return True
Then the HasItemPermission is called 6 times. A small problem with this example but huge when the custom permission contains database queries.
