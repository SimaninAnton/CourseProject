Choice field in Serializer class should be validate with (serializer.Serializer) even if the passing choice is not in the model choices. For example,
class MyModel(models.Model):

    class Gender:
        MALE = 1
        FEMALE = 2

        Choices = (
            (MALE, 'MALE'),
            (FEMALE, 'FEMALE'),
        )
    
    gender = models.IntegerField(
        _('gender'),
        choices=Gender.Choices,
        null=False,
        blank=False
    )



class Myserializer(serializer.Serializer):
    choice = serializers.ChoiceField(required=True, choices=MyModel.Gender.Choices)
    
    def validate_choice(self, value):
        if value in range(1,3):
            return value
There should be validate_choice() to custom validate the field.
