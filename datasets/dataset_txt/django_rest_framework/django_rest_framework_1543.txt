BooleanField serialization fails if fields source points to an unhashable, yet bool()-able value.
In my case I'm using django-bitfield to store flags on my model and I try to serialize a single flag with BooleanField(source='flags.changed), but bitfield represents flags with custom (unhashable) type.
As a workaround I pointed fields source to flags __bool__ method e.g. BooleanField(source='flags.changed.__bool__).
