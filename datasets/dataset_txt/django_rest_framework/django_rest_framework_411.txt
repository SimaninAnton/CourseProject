Use of built-in formats a little discouraging. Is not it better to write a special validator method for this purpose (e.g to_representation) ?
# In types.Type:

def __getitem__(self, key):
    value = self._dict[key]
    if value is None:
        return None
    validator = self.validator.properties[key]
    if hasattr(validator, 'to_representation'):
        return validator.to_representation(value)
    return value
This will make it easy to extend validators.
