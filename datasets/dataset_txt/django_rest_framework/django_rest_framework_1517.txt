Hello, I propose to add the ability to customize the get_object function in this way:
filter_kwargs = {self.lookup_field: self.get_lookup_kwarg(lookup_url_kwarg)}

def get_lookup_kwarg(self, lookup_url_kwarg):
        """
        Returns lookup kwargs
        """
        return self.kwargs[lookup_url_kwarg]
with the ability to override the get_lookup_kwarg() method in concrete viewset instead of:
filter_kwargs = {self.lookup_field: self.kwargs[lookup_url_kwarg]}
Current version seems to be not enough flexible.
