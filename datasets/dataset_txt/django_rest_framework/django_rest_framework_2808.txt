Follow http://django-rest-framework.org/api-guide/throttling.html#scopedratethrottle to set throttling.
For a APIView subclass with only a get method
Add to settings.py REST_FRAMEWOR:
    'DEFAULT_THROTTLE_CLASSES': (
        'rest_framework.throttling.ScopedRateThrottle',
    ),
    'DEFAULT_THROTTLE_RATES': {
        'category_list': '10/min',
        'country_list': '10/min'
    }
Subclass of APIView:
throttle_scope = 'category_list'  # scope = 'category_list' neither works
